# Intelligent Image Classification Engine (MobileNetV2)

An efficient, high-accuracy image classification system built with **TensorFlow** and **MobileNetV2**. This project was designed to automatically categorize cloud-stored imagery into five distinct functional types.



## 🚀 Overview
This engine uses **Transfer Learning** to classify images with over **91% accuracy**. By leveraging the pre-trained weights of MobileNetV2, the model can identify complex visual patterns while remaining lightweight enough for mobile and cloud integration.

### Classified Categories:
* **Documents:** Scans, PDFs-turned-images, and text-heavy photos.
* **Selfies:** Front-facing camera portraits.
* **Screenshots:** System-generated UI captures.
* **Low Quality:** Blurry, noisy, or low-resolution files.
* **Normal Photos:** Standard landscape or object photography.

## 📊 Dataset & Results
* **Total Images:** ~17,000 files.
* **Split:** 70% Training / 30% Testing.
* **Final Test Accuracy:** 91.06%
* **Optimization:** oneDNN custom operations enabled for faster CPU inference.
